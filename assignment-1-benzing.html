System Analysis - Congress.gov

Introduction

The information system I am going to analyze is the Congress.gov/ website. This website is used to serve both Congressional staffers and the everyday people. Congress.gov provides information on a multitude of Congressional publications, which include:

  • Legislation
  • Legislative texts
  • Nominations
  • The Congressional Record
  • Treaty documents
  • Slip Laws

When looking at this publications, congress.gov/ provides full texts, sponsrship information, and publication information. This allows these documents to be accessed and analyzed to the furthest extent.

Analysis

When looking at legislation it provides the Congressmember who sponsors the bill, which committee the bill gets referred to, dates of the legislative history of the bill, as well as the full text. Overall, it is a very large database and cannot be completely digested and analyzed in this one report.

There were a few things I thought were interesting about this database. One was that it included a long list of links to outside sources for Congressional staffers. These links can point to more condensed databases and information sources. Some of which is only accessible through Congressional networks. Some other things I thought were interesting were:

  1. Hyperlinks to other governmental resources - some only accessible by those on Congressional/private networks.
  2. A large range of search options that can help to narrow search results
  3. Also contains CRS reports without having to access crs.gov

      4. There are also both pros and cons to using Congress.gov

      Pros

        Provides access to a lot of data and information within one database
      • Spreads over both chambers of Congress
      • Provides resources for Congressional staffers, including direct resources for House staffers and Senate Staffers

      Cons

      • Can be confusing to first time users
      • Very daunting and intimidating when first visiting the site
      • Assumes the user has prior knowledge of the legislative process and Congress.

      Possible Improvements

      One improvement that can be made is to make the initial page less overwhelming by hiding the advanced search options or minimizing it to a smaller area on the site.
